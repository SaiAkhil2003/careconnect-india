# Real Provider Import Checklist

Use this checklist before importing or publishing real providers. Demo providers do not count as real provider coverage.

## Required Checks

- [ ] Provider name verified.
- [ ] City verified.
- [ ] Areas covered verified.
- [ ] Service types verified.
- [ ] Phone verified.
- [ ] Email or lead contact verified.
- [ ] Website checked if available.
- [ ] Source URL recorded.
- [ ] Source type recorded.
- [ ] Consent status recorded.
- [ ] Verification status is Approved.
- [ ] Consent has been received.
- [ ] No copyrighted website text copied.
- [ ] No fake verification.
- [ ] No duplicate provider slug.
- [ ] Admin reviewed.
- [ ] `is_active` set correctly.
- [ ] `is_verified` set correctly.
- [ ] `listing_tier` set correctly.
- [ ] `service_types` use only app-supported service keys.
- [ ] Audit notes retained in the research tracker or review archive.

## Publication Rule

Publish only providers that have verified details, recorded consent, and admin approval. If any required item is uncertain, keep the provider inactive until review is complete.
